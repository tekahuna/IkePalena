# Security Policy

This repository contains **public release materials only**.

## Reporting a vulnerability
If you believe an uploaded release asset is corrupted or mismatched:
- Provide the GitHub Release tag and asset name
- Provide the expected SHA256 (from the manifest)
- Provide the computed SHA256 you observed

Do **not** request internal vault, CAS, ledger, or receipts.
Those are not part of the public transport layer.
